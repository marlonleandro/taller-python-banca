¿Por qué Python en el Negocio?

El valor ejecutivo de adoptar Python para profesionales financieros:

  • ⚡ Eficiencia:  Reduce drásticamente las horas manuales en reportes y conciliaciones.
  • ✅ Calidad:  Minimiza errores humanos, garantizando mayor trazabilidad y repetibilidad.

Fundamentos · Bloque 1

Variables y Tipos de Datos

  • ¿Qué es una Variable?  Una "caja" con nombre donde guardamos información temporal para usarla después.
  • Tipos Comunes:  Números (enteros/decimales), Texto (palabras), Fechas (calendario) y Booleanos (verdadero/falso).
  • Ejemplos Bancarios:  saldo = 15000.50 · segmento = "Pyme" · es_moroso = False
  • Buenas Prácticas:  Usar nombres claros como tasa_interes_anual, mantener unidades consistentes.
  • ¿Para qué sirve?  Para crear filtros automáticos, calcular KPIs masivos y aplicar reglas de negocio sistemáticamente.

Conceptos Clave

Listas [ ]: Colecciones ordenadas. Ideales para agrupar múltiples registros — clientes, transacciones.

Diccionarios { }: Estructuras clave-valor. Cada fila de Excel es un diccionario.

Usos Prácticos en Banca

Construir la "Vista 360" del cliente guardando todos sus productos en un diccionario. Filtrar transacciones sospechosas con list comprehensions.

Mini Caso Práctico

Caso Integrador: Análisis de Cartera

# 1. Cargar datos de Excel
import pandas as pd
cust = pd.read_excel("clientes.xlsx")
loans = pd.read_excel("prestamos.xlsx",
        parse_dates=["due_date"])

# 2. Filtrar clientes activos
cust_active = cust.loc[
    cust["status"] == "Active",
    ["id", "segment"]]

# 3. Calcular KPIs a nivel cliente
ref_date = pd.Timestamp("2026-04-20")
loans["dpd"] = (ref_date - loans["due_date"])\
             .dt.days.clip(lower=0)
kpis = (loans.groupby("customer_id")
        .agg(balance=("balance","sum"),
             dpd_max=("dpd","max"))
        .reset_index())
kpis["is_past_due"] = kpis["dpd_max"] > 0

# 4. Unir datos y resumen por segmento
base = cust_active.merge(kpis,
        left_on="id", right_on="customer_id")
seg_summary = (base.groupby("segment")
    .agg(portfolio=("balance","sum"),
         past_due_rate=("is_past_due","mean"))
    .reset_index())

# 5. Generar insights
top_seg = seg_summary.sort_values(
    "portfolio", ascending=False).head(3)
alert = seg_summary.loc[
    seg_summary["past_due_rate"] > 0.08,
    "segment"].tolist()

🎯 Objetivo

Integrar flujo completo: carga de datos, filtros, cruce y cálculo de KPIs para obtener insights de negocio accionables.

📋 Paso a Paso

  1. Cargar archivos Excel de base.
  2. Dejar solo cartera activa relevante.
  3. Calcular mora (días desde vencimiento) y consolidar.
  4. Unir perfil con KPIs y agrupar por segmento.
  5. Extraer top portafolios y alertas de mora.

📈 Resultados Esperados

Visión clara de cartera por segmento. Detección temprana de mora alta (>8%). Script automatizable para reporte diario.
